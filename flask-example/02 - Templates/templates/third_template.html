On this page, you will be able to see the effects of passing Python variables (similar to "name" in the previous template) and using the {% raw %}{%%}{% endraw %} statements delimiters to perform if-statements on this variable inside the template.

The number is {{number}}.

{% if number > 50 %}

Your number is above 50.

{% else %}

Your number is below 50.

{% endif %}

This is the result of a conditional.
